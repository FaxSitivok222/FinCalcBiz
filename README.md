# FinCalcBiz

Простий статичний вебсайт — калькулятор щомісячного платежу для бізнес-кредиту.

## Що в комплекті
- `index.html` — головна сторінка
- `calculator.html` — сторінка з калькулятором
- `about.html`, `contacts.html`, `faq.html`
- `css/style.css` — стилі (бежево-карамельна палітра)
- `js/main.js` — логіка калькулятора
- `README.md` — цей файл

## Як працює калькулятор (коротко)
- Вводите: **сума кредиту**, **річна ставка (%)**, **термін у місяцях**.
- За потреби — **комісія** (фіксована сума) та **податок на комісію (%)**.
- Комісія та податок додаються до суми кредиту перед розрахунком щомісячного платежу.
- Формула: ануїтетний платіж — `A = P * r / (1 - (1+r)^-n)`.

> Якщо потрібно інше правило (наприклад, комісія віднімається окремо або комісія як % від суми), повідомте — я зміню код.
